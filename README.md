## 功能

改變 Ed 中 Run / Stop 按鈕的顯示狀態。

## 安裝方式（以 Google Chrome 為例）

在[Releases 頁面](https://github.com/skylee03/ed-helper-crx/releases)下載最新版本的 zip 檔案。

進入[Chrome 擴充功能設定](chrome://extensions/)，啟用開發人員模式。

將剛才下載的 zip 檔案拖入即可。

# 使用方法

點選擴充功能圖示即可改變 Run / Stop 按鈕的顯示狀態。
